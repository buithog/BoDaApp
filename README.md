# BoDaApp
App android of BO DA pagoda

Application for Bo Da Pagoda relic area includes functions of viewing news, listening to explanations, scanning QR,....

# Technology
 language : Java 17;
 
 IDE : Android studio;
 
 API 34;

# Set up
  Go to file network_security_config.xml to config IP;
